# 🚀 粵語 AI 接線生系統 - 產品介紹

## 核心優勢
✅ **完全廣東話** - 自然對話，無機械感  
✅ **10秒回應** - 客戶不會掛線  
✅ **智能記憶** - 自動避免重複提問  
✅ **24/7 無休** - 節省人力成本  

## 適用行業
- 美容院、髮廊
- 餐飲（外賣、訂座）
- 健身房、瑜伽館
- 診所、牙醫
- 酒店、民宿

## 投資回報 (ROI)
```
傳統：1名接線生 ≈ ¥15,000/月 × 12 = ¥180,000/年
AI 系統：¥5,000/月 × 12 = ¥60,000/年
節省：¥120,000/年 ✅
```

## 技術堆棧
- 前端：HTML5 + Web Audio API
- 後端：Flask + Python
- 語音識別：Azure Speech Services (粵語)
- 語言模型：Google Gemini 1.5 Flash
- 文字轉語音：Azure TTS (zh-HK-HiuMaanNeural)

## 部署方案
- 雲端版：AWS/阿里雲 + HTTPS (推薦)
- 本地版：Docker 容器化
- API 版：支援第三方系統集成
